\section{Motivation}
\begin{frame}{Motivation}
  \begin{itemize}
    \item Football broadcast analysis is useful for coaching, scouting, and match review.
    \item The challenge is to robustly detect small objects, maintain identities, and derive usable statistics.
    \item This project uses a deep detector plus temporal post-processing to turn video into match metrics.
  \end{itemize}
\end{frame}

\begin{frame}{Method Summary}
  \begin{itemize}
    \item Detection: fine-tuned YOLOv8 for ball, player, goalkeeper, referee.
    \item Tracking: ByteTrack-style persistent IDs.
    \item Team assignment: jersey colour clustering on player crops.
    \item Ball recovery: interpolation + ROI-based recovery for short gaps.
    \item Camera motion: optical-flow-based compensation when enabled.
  \end{itemize}
\end{frame}

\section{Discussion}
\begin{frame}{Discussion and Error Cases}
  \begin{itemize}
    \item The fine-tuned model learns football-specific visual cues and improves small-object recall.
    \item Interpolation acts as a temporal prior and fills short ball gaps.
    \item Remaining failure cases: airborne ball, motion blur, strong occlusions, goalkeeper ambiguity.
    \item Camera compensation and team assignment mainly improve downstream analysis coherence.
  \end{itemize}
\end{frame}

\begin{frame}{Limitations and Future Work}
  \begin{itemize}
    \item Single broadcast viewpoint and limited number of videos.
    \item Ball speed metrics depend on calibration and metric scale.
    \item Future work: better small-object detector, temporal model, pitch calibration, multi-view analysis.
  \end{itemize}
\end{frame}
